经验
科研, 论文

文献检索

在哪找文献

  • 中文论文

    • 1.知网（推荐）

      • 博硕论文
      • 可通过云大图书馆免费下载论文

    • 2.百度学术

      • 知网找不到的中文论文来这找

  • 英文论文

    • 谷歌学术
    • 可通过谷歌学术镜像访问，建议访问原网站

  • 其他渠道

    • 1.综述论文下面引用的那些文献

    • 2.GitHub、公众号、知乎、blog上的论文列表

      https://github.com/THUNLP-MT/MT-Reading-List#meta_learning

谷歌学术检索技巧

  • 关键词搜索

    • 根据研究问题，想出几个关键词；用空格分开组合搜索

  • 按日期筛选

    • 查找近年来的最新文献

  • 高级搜索

    • 精确搜索，结果更准确

  • 相关文章

    • 找到类似的文献

  • 被引用次数

    • 看其他文献怎么评论该文献

文献管理

Endnote基本使用

软件包分享，需要校内IP下载

  • 从谷歌学术导入文献

  • 关联PDF文件

    • 以后读文献、高亮文献都从Endnote打开，统一管理

  • 文献分类

    • 建立索引，方便查找
    • 集中时间看同一类的论文，便于比较

  • 一句话总结这篇文献，方便下次阅读

文献同步

  • 登录Endnote账号同步

  • 百度网盘工作空间同步

    • 同步Endnote存文献的那个文件夹

  • GitHub Desktop

文献阅读

知云划线翻译

  • 选有道或者谷歌医学引擎
  • 阅读的过程中高亮重要句子

阅读顺序：摘要、结论、引言、讨论、结果和方法

先泛读，然后选择重要的文献精读

  • 在Endnote分组：重要/有些重要/不重要

文献笔记

为什么要记笔记

  • 1.需要重复阅读某篇文章时。提高阅读文献的效率，减少重复工作
  • 2.读完某篇文章的思考和收获，文章有哪些点可以借鉴
  • 建议重要的文章记下笔记，不重要的高亮一下就行了

建立知识库

  • Notion（推荐）
  • Excel表格（推荐）
  • GitHub repo

将高亮的句子全部翻译为中文

  • 总结与归纳，记下要点：根据句子翻译提炼重点
  • 同时记录读这篇文章的体会、想法，或者与其他文献的关联比较，或者该文献的亮点与不足等

每个分类单独开一个文件或表格记录笔记

  • 便于比较和分析
