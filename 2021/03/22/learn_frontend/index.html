前端学习路线和学习方法总结

Created2021-03-22|Updated2021-03-23|前端
|Post View:

阶段1：HTML，CSS，JavaScript三件套入门

这个阶段的重心是入门，为后面打好基础，以及良好的编码风格。

学习路线

参见GitHub上的frontend roadmap，照着这个学没错的。把必学的先学

学习方法

视频入门+文档补充+博客为辅+面经和书籍打基础

每学一个阶段，记录下笔记。看看面经记笔记。写的博客要有深度性

学习内容

  • HTML
  • CSS
  • JavaScript
  • ES6
  • jQuery
  • Ajax
  • Git/GitHub

学习时间

2~3个月时间完成该阶段，因为还有其它事情要做
不需要完全都会，学习JS还有很长的路要走

学习任务

  • 掌握HTML，CSS，JavaScript基础
  • 在不依赖前端框架的前提下，使用原生代码完成一个网站的开发。

学习要点

  • 关于视频：视频要跳着看，会了的就不用看了，看看文档就OK了；如果可以的话最好1.25或1.5倍速看；记录下哪个分P的视频值得重看，方便再刷。
  • 学视频和看书要跟着敲一下代码，只看不做是不行的。听完之后要做一下笔记吸收消化。
  • 写代码的时候会出现各种问题，要记录下这些问题，出现的原因，解决的办法等等。
