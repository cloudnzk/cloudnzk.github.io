熟练使用快捷键能提高开发效率

快捷键 用处
ctrl + 鼠标单击 跳转到变量、类或方法名的声明
ctrl + / 单行注释
ctrl + shift +/ 多行注释
ctrl + d 删除选定的行
ctrl + z 撤销
ctrl + o 当前代码的结构
ctrl + f 在当前代码中搜索
ctrl + h 全局搜索
ctrl + shift + r 快速搜索，可指定搜索范围
alt + 左/右 跳转到下一个、前一个编辑的选项卡
alt + 上/下 向上/下调换两行代码
tab 往后移动4个空格
shift + tab 往前缩进4个空格
